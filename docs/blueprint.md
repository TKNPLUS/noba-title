# **App Name**: Thumbnail Title Guesser

## Core Features:

- Thumbnail Display: Display a YouTube thumbnail from the CSV data.
- Title Guess Input: Allow users to input their title guess.
- Title Matching: Compare the user's guess with the actual title and provide feedback.
- Score Tracking: Track the user's score and display it.
- Data Loading: Load YouTube video data from a CSV file. The app should be able to dynamically refresh the CSV when updated (the MVP implementation might involve manually updating a JSON file).

## Style Guidelines:

- Primary color: Vibrant blue (#29ABE2), reflecting YouTube's brand identity.
- Background color: Light gray (#F0F0F0), providing a neutral backdrop.
- Accent color: Bright orange (#FF8C00), used for feedback messages (correct/incorrect) and score display.
- Font: 'Inter', sans-serif, for a clean, modern, readable design; good for headlines and body.
- Simple, recognizable icons for score and feedback.
- Clean and centered layout with ample spacing.
- Subtle transitions for feedback messages and score updates.
